Using NestKey

To add a new credential, click the “Add Credential” button.

Viewing a Credential

Click the eye icon in the “Notes” column to view any notes you saved with the credential.

Generating Strong Passwords

Use the password generator dialog (the “Generate” button) to create a random, strong password.

Copying Passwords

Click the “Copy” button next to a password in the table to copy it to the clipboard.

Searching

Use the search box at the top of the vault to filter credentials by site, username or notes.

Changing Your PIN

Click the “Change PIN” button on the lock screen and follow the prompts.

Security Considerations

This prototype stores the encryption key in memory for the whole session. For a production‑grade app you should add:

  • Exponential back‑off lockout (already added).
  • Secure backup / restore (encrypted export).
  • Hardware‑backed key storage (e.g., OS keychain, TPM).
  • A full third‑party security audit.
